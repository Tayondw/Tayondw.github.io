Overview

*The project is currently being updated to include new features such as the Google Maps API, FastAPI, and Mental Treat API. These additions will provide real-time data, enhancing user access and efficiency.

An original and intuitive application that intended to connect males users with mental health issues. Users can join groups or attend events based off their associated tag - which is their mental health issue they are seeking to improve. The site also leverages the BetterHealth API to allow users to access therapist's locations so they can reach out to and/ or meet. The Google Maps API is utilized for a better and more accurate access of group, event, and venue location.
